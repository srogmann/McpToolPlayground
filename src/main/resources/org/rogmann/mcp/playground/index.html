MCP Server Simulation

How to Use This MCP Simulation

Welcome! This tool lets you simulate a Model Context Protocol (MCP) server.

Step-by-Step Guide

  1. Enter Your Name: Filled by the server.
  2. Define a Tool: Give it a title and description (e.g., "get_weather").
  3. Define Input Properties: What inputs does your tool need? (e.g., "Place")
  4. Start the MCP Server: Click the button to launch the simulated LLM interaction.
  5. Respond to Tool Calls: When the LLM sends a request, fill in the value and click "Send Response".
